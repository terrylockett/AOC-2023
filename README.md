## Usage

#### Build
`./gradlew build`

#### Run
`./gradlew 2023:day01:run`

#### Lint
`./gradlew 2023:day01:spotlessApply`

#### Test
`./gradlew test`

#### Create new Module
`./gradlew newModule -PmoduleName=2023:day02`
